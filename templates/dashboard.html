📈 Project Overview

Smart Lender is a Machine Learning application that predicts whether a loan should be approved or rejected. The project compares four algorithms:

  • Decision Tree
  • Random Forest
  • K-Nearest Neighbors (KNN)
  • XGBoost

The best performing model is selected automatically and used for loan prediction.
