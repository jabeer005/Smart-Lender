🏦 About Smart Lender

Project Overview

Smart Lender is a Machine Learning based web application that predicts whether a loan should be approved or rejected based on applicant details. The system helps banks and financial institutions make faster and more accurate loan approval decisions.

Problem Statement

Manual loan verification is time-consuming and may lead to incorrect decisions. Smart Lender automates this process using Machine Learning algorithms.

Objectives

  • Predict loan approval using Machine Learning.
  • Reduce manual verification time.
  • Improve decision accuracy.
  • Provide instant loan approval prediction.
  • Deploy the application on IBM Cloud.

Project Workflow

  1. Collect Loan Applicant Data
  2. Preprocess the Dataset
  3. Train Machine Learning Models
  4. Compare Accuracy
  5. Select Best Model
  6. Save Model (.pkl)
  7. Create Flask Web Application
  8. Predict Loan Status
  9. Display Result
