# 💰 Smart Lender

An AI-powered Loan Approval Prediction System built using **Python, Flask, Machine Learning, HTML, CSS, and JavaScript**.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/jabeer005/Smart-Lender.git
```

Move into the project

```bash
cd Smart-Lender
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open in browser

```
http://127.0.0.1:5000
```
